# Tailwind React Integration

This project is a React app created with Vite and configured with Tailwind CSS.

## Setup Steps
- Created project with Vite
- Installed Tailwind CSS
- Configured Vite with Tailwind plugin
- Imported Tailwind in index.css
- Ran `npm run dev` to verify setup
